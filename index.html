List Elements

List consist of two types.

  • unordered lists
  • ordered lists
List are started with either an unordered list tag <ul>, or an ordered list tag <ol>. The items within the list are list items which are wrapped in a list item tag aka <li>.
each tag will need a closer <ul></ul>, <ol></ol>, <li></li> You can also embed lists within other lists like so.

  2. b

Images!!

Next we learn about images(img). To add an image into you're HTML file you need to have a source or a file to refer to. This will be shown as src later down the line.
<img (for image) src(source) ="(link to image here)"
alt (for screen readers)= "(words for screen readers to read usually something about the image itself)>
so in the grad scheme it'll look something like this:
<img src = "randomimageofsomething.jpeg"
alt=" describe random image">

Misc. Things to remember:

Somethings to remember include, inline and block elements. Such as spans and divs respectively.
a div is a block element that takes everything that is within the opening and closing tags and allows them to be stylized as a group. Almost creates a block around the elements to show they are grouped.
Spans work similarly to divs but are instead in line elements which means they still group but try to take up as little space as possible, imagine a grocery list ith highlighted elements. the highlighted elements would be the spans.

Entity codes.

Entity codes are place holders for things in html that serve a purpose in the coding side of things such as < and >. Entity codes allow us to use special characters such as ® and ☃ that are otherwise not available.
A few simple entity codes are &lt; for the < sign and &gt; for the > sign also &amp; for &
