Tables
Tag Use
Table This tag is what starts our table
TH this is the table heading tag, starts automatically bolded. REQUIRES TR
TR This is the tabel row tag, everything inh the table will be nested into a table row.
TD This is the Table cell tag. This is each induvidual cell in a row.
Thead The header wrap. Put all headers in a thead tag to make it easily distinguishable.
Tfoot Same thing as thead except this is more than like going to be a total or sum.
Tbody Wrap the main information of a table into a tbody tag
Colspan makes a td or th take up multiple columns colspan="2"
Rowspan make a td or th take up multiple rows rowspan="4"
