Cascading Style Sheets!

Now that we've covered a vast majority of html we will now be moving onto Cascading Style Sheets or CSS for short.

What is CSS?

CSS is a language, not really a programming language but a language non the less. It allows us to make changes to HTML! CSS is a core component of most websites. Allowing us to move and color different things.

The typical syntax for CSS or RULES look something like this:

selector {
property: value;
}

Several examples of this are
h1{ color:purple; } this allows us i make all h1s in our code the color purple.
img{ width:200px; height:100px; }while this allows us to change the size of an image.
etc.
