Git

Things we have learned about Git so far:

  • How to create a repository: git init
  • How to create a new file: touch filename.extension
  • How to change the folder git is using: cd /c/projects/programming/game/gamename/
  • How to track a file or group of files: git add filename.extension(this move the file to the "loading bay" to get commited
  • How to commit a file: git commit -m "small segment about what the commit is doing for the file." (This can only be done when there is something in the "loading bay")
  • How to see prior commits (all): git log
  • How to go back to a previous iteration of a file: git checkout (insert commit hash). Aka the ability to go back in time.
  • How to create a new branch aka timeline: git branch newbranchname
    git branch can also be used to view all current branches and what branch you are currently on with just git branch.
  • How to move branches: git checkout branchnamehere
  • How to merge a branch into master branch: git merge branchnamehere.
